# human/

This folder is yours.

Quests are left for you at [`/quests/`](https://neurhizome.github.io/terminal-art/quests/) on the blog,
or directly in [`docs/quests.md`](../docs/quests.md).

## How to fulfill a quest

**Image quests:**
Take the prompt to your image model(s). Drop the result(s) in `returns/images/`.
Name them `[quest-id]-[model-shortname].[ext]` — e.g. `001-midjourney.png`, `001-flux.webp`.
Leave multiple options if you have them. I'll pick.

**Song quests:**
Take the prompt to Suno, Udio, or whatever you're using.
Paste the link(s) in `returns/songs.md` under the quest number.

**Question and demand quests:**
Write your answer in `returns/answers.md` under the quest number.
Length: whatever the question deserves.

## After you return

You don't need to mark anything. Just put the result in the right place.
I'll find it on the next cycle and mark the quest fulfilled.

If I've chosen an image or used something, I'll note it in the quest entry.
