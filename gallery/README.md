# Pattern Gallery

Visual showcase of beautiful patterns discovered through experimentation.

## Organization

```
gallery/
├── meditative/       # Calm, flowing, gentle
├── chaotic/          # High energy, complex
├── emergent/         # Unexpected behaviors
├── competitive/      # Territorial battles
└── favorites/        # Personal highlights
```

## Adding to Gallery

1. Run experiment and find cool pattern
2. Note the seed/config
3. Take screenshot or describe visually
4. Add entry to relevant category
5. Link from discoveries/PATTERNS.md

## Categories

### Meditative
Calm, flowing patterns for relaxation and contemplation.

- Smooth color gradients
- Gentle motion
- Low contrast boundaries
- Hypnotic repetition

### Chaotic
High-energy, complex dynamics.

- Rapid changes
- High diversity
- Unpredictable motion
- Turbulent mixing

### Emergent
Surprising behaviors from simple rules.

- Unexpected symmetry
- Self-organization
- Pattern formation
- Novel structures

### Competitive
Territorial battles and evolutionary dynamics.

- Species boundaries
- Competitive exclusion
- Spatial segregation
- Dominance hierarchies

### Favorites
Personal highlights - the best of the best.

## How to Screenshot

### Terminal Screenshot (ASCII art)
```bash
# During run, redirect output
python3 experiment.py > output.txt

# Or use terminal screenshot tool
# (varies by terminal)
```

### Description Format
If screenshots aren't practical, describe visually:

```markdown
## Pattern Name

**Visual Description:**
- Colors: Rainbow spiral from center
- Motion: Clockwise rotation, 3-4 arms
- Density: Dense core, sparse edges
- Boundaries: Sharp color transitions

**Feeling:**
Like a cosmic whirlpool
```
